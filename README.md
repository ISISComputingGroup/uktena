# uktena

This is the python _distribution_ deployed to instruments and developer PCs, generally in `C:\instrument\apps\python3\`. 

It contains all the dependencies for anything deployed in the EPICS tree as well as providing libraries (including [`genie-python`(https://github.com/ISISComputingGroup/genie)] for user scripts.

For information on building it locally, see [this page of the dev wiki](https://isiscomputinggroup.github.io/ibex_developers_manual/system_components/python/Building-and-installing-uktena.html#building-and-installing-uktena)
